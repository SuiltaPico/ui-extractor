服务于边缘计算的纯文本模型的 UI 提取器。
给出所有 ui 元素的坐标。

## UI 树识别

灰度 → Canny → 膨胀 → 闭运算 → 轮廓层级（Suzuki-Abe），构建轴对齐 UI 树。

## 文本识别

PaddleOCR PP-OCRv5 mobile（ONNX，via `oar-ocr`）。

## 图标识别

从布局树筛选近似正方形的叶子容器作为候选，crop 后经 MobileCLIP2-S0 向量嵌入，与 MDI 预计算索引做余弦检索。

匹配流程：

1. 从 layout 候选区域 crop 出图标
2. 预处理为 256×256 白底黑 icon RGB
3. MobileCLIP2-S0 ONNX 嵌入 → 与 `assets/mdi/embeddings.bin` 余弦检索
4. top-10 候选 + mask IoU rerank；`cosine ≥ 0.72` 则命中
5. 未命中保留 `container`；后续 potrace 生成 SVG path 交给 AI

详细说明见 [icon-matching.md](./icon-matching.md)。MDI 资源准备见 [mdi-icons.md](./mdi-icons.md)。
